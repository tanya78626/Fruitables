🍎 Fruitables – Online Fruits & Vegetables Store
📌 Overview

Fruitables is a Django-based web application that allows users to browse, select, and purchase fruits and vegetables online. The project focuses on simplicity, usability, and basic e-commerce functionality.

🚀 Features
User Registration & Login
Browse Fruits and Vegetables
Add to Cart
Checkout System
Online Payment Integration (Razorpay)
Order Placement
🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap
Backend: Django (Python)
Database: SQLite
Payment Gateway: Razorpay


📌 Future Improvements

User profile management
Order tracking system
Responsive UI improvements
